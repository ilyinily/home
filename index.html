Finding piece in PHP

So today we had a call about this nasty incident happened with starting many many jobs. One of the ideas was to limit user's ability to make error easily. We did ask the developer to update the app code, but it takes some time.

So I tried to apply my low skill in HTML and find the place where the edit can actually be done. Found in page description the PHP script that needs to run, then spent a few minutes looking for its location in the server in sandbox environment.

It was a success, and then there was a question where exactly is the faulty place. Copied PHP script to VS Code and looked inside. Found something that likely was the spot. Googled for how to use Vim and after a few unsuccessful attempts was able to save the result.

Then I thought same could be done for submitting jobs as well, and tweaked this bit, too.

Next up was the need to remove the pre-stated tickboxes. This was a bit more difficult as the PHP script was longer. Found a place where likely the checkboxes can be unchecked and replaced the word.

Thought that in reality I don't need to put the word 'unchecked' but rather remove it completely. But didn't have time. Will try Monday.

Felt pride of how successful it was. I now fear it less.
