Reading from files

Today I have picked up a couple functions to read content from files. One is for reading object content, other for arrays. With arrays, it turned out to be tricky - for some reason when removing the \r characters from the lines, the \n characters remained in their place with an added \ before them. So had to search for a way to get rid of this and found a way with regular expressions. Not sure entirely how it works however, need to learn more.

In the end, however, the reading from files turned out to work well. Hopefully I'll be able to make it massive to avoid creating a variable or constant for every single bot dialog.
